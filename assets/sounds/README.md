# תיקיית צלילים

בתיקייה זו יש להציב את קבצי הצלילים הבאים:

## צלילי UI:
- `button_click.mp3` - לחיצת כפתור
- `button_hover.mp3` - מעבר עכבר על כפתור
- `page_transition.mp3` - מעבר בין דפים
- `dropdown_open.mp3` - פתיחת תפריט נפתח
- `dropdown_close.mp3` - סגירת תפריט נפתח

## צלילי הצלחה/שגיאה:
- `success.mp3` - הצלחה
- `error.mp3` - שגיאה
- `warning.mp3` - אזהרה
- `info.mp3` - מידע

## צלילי התראות:
- `new_request.mp3` - בקשה חדשה
- `new_message.mp3` - הודעה חדשה
- `request_accepted.mp3` - בקשה התקבלה
- `request_completed.mp3` - בקשה הושלמה

## צלילי אימות:
- `login_success.mp3` - התחברות מוצלחת
- `logout.mp3` - התנתקות
- `registration_success.mp3` - הרשמה מוצלחת

## צלילי תשלום:
- `payment_success.mp3` - תשלום מוצלח
- `payment_failed.mp3` - תשלום נכשל

## מוזיקת רקע:
- `background_music.mp3` - מוזיקת רקע (אופציונלי)

## המלצות:
- השתמש בפורמט MP3 לאיכות טובה
- שמור על קבצים קטנים (עד 100KB לכל צליל)
- השתמש בצלילים קצרים (1-2 שניות)
- ודא שהצלילים מתאימים לקהל היעד
